# Labyrinttiratkoja
Tiralabra alkukesä 2020

## Aihe    
Verkot ja polun etsintä; miten löydetään tehokkaasti nopein/lyhin reitti labyrintistä ulos.    
## Ohjelmointikieli  
Java  
## Työn laajuus  
Ohjelma, joka luo labyrintin ja sitten ratkaisee sen. Sisältää yksinkertaisen käyttöliittymän, jossa voi nappia painamalla luoda labyrintin ja nähdä, minkä ratkaisun ohjelma tälle satunnaisesti luodulle labyrintille on löytänyt.
