## Käyttöohjeet

Huom! Ei toimi vielä

Labyrinttiratkoja on Java-sovellus

1. Kloonaa sovellus

git clone https://github.com/jaanan/labyrinttiratkoja.git
cd labyrinttiratkoja

2. Kokoa

javac -sourcepath ./src -d bin src/labyrintti/Main.java

3. Aja

java -classpath ./bin labyrintti.Main
